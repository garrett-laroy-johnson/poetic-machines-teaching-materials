NIDUS is a procedural generative 3D space by WEBGL shader.

Signed distance function and different noises are used to create the dynamic shape transforming and the terrain deforming.

By merging regular and irregular forms, I want to create a space that is between organic and inorganic, and has a atmosphere between mysterious and cyberpunk.
