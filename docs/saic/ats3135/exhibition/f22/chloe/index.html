RIG

Random Image Generator

RIG (Random Image Generator) (2022) is an in-progress custom library that generates random images based off of a specified parameters set by the user.

RIG does a search through Commons Wikimedia and grabs images off the site for creators to use in their p5.js sketches.

How to Use: Once the library is imported into your p5.js sketch environment, creators can use the function RIG().then(function(result) {};

Here is an example sketch below:

let frog_img;

function setup() {
RIG("frog", 1).then(function(result) {
frog_img = result;
});

createCanvas(400, 400);
}

function draw(){
background(220);

if (typeof frog_img != "undefined") {
frog_img.createImage(50, 50);
}

}

First in the setup() function we want to use or RIG() function and use the .then action to set our random generated image result to be a set to a variable we can use in the draw() function

The RIG() function requires two parameters which are a text entry for the random image to be based off and number value to change the random image.

Once we have our RIG() result set to a variable we will then want to draw our image using an if statement to check if the variable has been set yet. After the result has been sent then we can use the .createImage function which can take a x and y parameters for canvas placement.
