p5*js 101 teach-in

The tired yet wired adage goes: if you can teach it, you can learn it. Or something like that. Your assignment is to work alone or with an assigned partner on a fundamental p5*js concept area in preperation to teach it to the rest of the class.

You may draw on any materials you like for this assignment. The Getting Started textbook is highly recommended; I have indicated some example numbers below to get you started. You can also check out my p5*js intro course website which is organized by these concept areas. Each page has links to video tutorials which may be helpful.

You should prepare:

  1. a 10-20 minute tutorial for the rest of the class. Aim to cover as many of the coding functions and concepts listed below as are reasonable within that time limit.
  2. a p5*js sketch "cheat sheet" that shows all the concepts and functions in action. Label these concepts and functions with a JS comment. The visual appeal of what is drawn in this "cheat sheet" are *not* important. Aim instead to create an instructive document with pedagogy in mind.

p5*js 101 presentations pt. 2:

  • movement
    • frameRate() [ex. 8-2]
    • moving a shape with the additive updater +=
    • using if statements to create wrap around + bouncing rules [ex. 8-4, 8-5]
    • random() + movement [ex 8-7, 8-8, 8-9]
  • functions
    • use a function to quickly draw multiple complex shapes (> 2 simple 2D shapes) using a function [ex 9-3, 9-4, 9-5, 9-6]
  • JS objects
    • properties (vs variables)
    • methods (vs functions)
    • composing an object [ex. 10-1]
    • creating many objects [ex. 10-2]
  • arrays
    • array as holder of variables [ex. 11-3] + filled w. for loop
    • assigning and declaring arrays [ex 11-4, 11-5]
    • accessing contents of arrays using [] notation [ex. 11-6]
    • using for loops to manage many JS objects [ex. 11-9]
